# Spassmonopoly-Deluxe
Spassmonopoly-Deluxe ist ein client-serverbasiertes Multiplayer-Spassspiel mit Lobby-System. Spieler treten über den Browser bei, der Server verwaltet Spielzustand, Würfel, Züge und Spiellogik in Echtzeit. Fokus: synchrones Gameplay, klare Backend/Frontend-Trennung und skalierbare Struktur.
